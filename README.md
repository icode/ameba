# 一个务实的框架
> 当你看到这个的时候，你应该明白这是国产的框架


## Ameba(阿米巴/变形虫)全栈的
1. 一些工具
2. 一个UI库
3. 一些模块
4. 一层架构（没有service，没有DAO，没有控制器）
5. 一点敏捷
6. 一个配置文件


## Ameba(阿米巴/变形虫)不优雅的
> ameba目标是快速开发，减少痛苦，好用

1. 不小，挺大
2. 没有高超的设计
3. 没有流弊的功能
4. 没有魔术
